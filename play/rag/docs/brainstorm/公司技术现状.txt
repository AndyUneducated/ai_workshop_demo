公司技术现状

一、基础设施

当前技术栈：后端 Python (FastAPI)，前端 React + TypeScript，数据库 PostgreSQL 14，缓存 Redis 7，部署在阿里云 ECS 上。
内部已有 GitLab CI/CD 流水线，容器化程度约 60%，剩余 40% 仍为裸机部署。
团队共 23 人：前端 6 人，后端 9 人，运维 3 人，产品 3 人，测试 2 人。

二、现有知识库系统

公司目前使用 Confluence 作为知识库，共有 4327 篇文档，涵盖产品需求、技术文档、运维手册、会议纪要四大类。
文档格式以 Markdown 和 HTML 为主，少量 PDF（约 180 份）。
员工对 Confluence 搜索的满意度调查：仅 23% 表示"能快速找到需要的信息"。
每月平均搜索量 12000 次，其中约 35% 的搜索未返回有用结果。

三、AI 相关经验

去年 Q4 曾尝试用 ChatGPT API 做过一个问答原型，但因回答准确率不足 40%（未接入内部知识库）而搁置。
团队中 3 名后端工程师有 LangChain 使用经验，1 名前端工程师熟悉 Vercel AI SDK。
公司已购买 OpenAI API 额度，月预算上限 2000 美元。
